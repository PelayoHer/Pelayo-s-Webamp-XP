Guía de Usuario - Winamp XP

Gestión de Audio

Para iniciar la reproducción, arrastre archivos de audio en formato MP3 directamente sobre la interfaz del reproductor o dentro del área de la lista de reproducción (Playlist).

Personalización de Interfaz (Skins)

  • Selector Directo: Utilice el menú desplegable en la esquina superior derecha para alternar entre las apariencias preconfiguradas.
  • Carga Manual: El sistema soporta la importación dinámica de skins mediante el arrastre de archivos con extensión .wsz sobre el reproductor principal.
  • Repositorio Externo: Puede adquirir una vasta colección de apariencias adicionales en el portal oficial skins.webamp.org.

Visualizaciones Milkdrop

El motor de visualización Milkdrop se encuentra configurado para alternar de forma aleatoria entre más de 600 estilos con cada cambio de pista. Para fijar un estilo específico, utilice el selector secundario ubicado en la barra de controles superior.

Controles Avanzados

Mediante el clic derecho en cualquier sección del chasis del reproductor, podrá acceder a menús contextuales para la gestión detallada de la reproducción, ecualización y opciones internas del visualizador.
